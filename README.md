# E-Document Integration Workshop

Hands-on workshop for building E-Document integrations in Business Central.

## What's Included

- **SimpleJson Format** - AL extension for exporting/importing documents as JSON
- **Directions Connector** - AL extension for sending/receiving documents via REST API
- **Server** - Python Flask API for document queueing
- **Workshop Guide** - Step-by-step exercises with solutions

## Quick Start

1. Review the [Workshop Guide](./workshop/WORKSHOP_GUIDE.md)
2. Follow the setup instructions
3. Complete the 4 exercises:
   - Export documents to JSON
   - Send documents via API
   - Receive documents from API
   - Import documents to Business Central
